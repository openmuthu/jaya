Welcome to आगमान्वेषिणी.

You can browse and install new granthas using Settings > Download/Update Content option

You can use '#' tags to limit your search to specific granthas

Eg: To search for 'hanumaan' in 'sarvamUla' you can type the following.

hanuma #sarva

You can also use other tags like #nArada, #purANa, #sudhaa, #giita, #bhaarata, #vyaaka, #kOsha, #hkms etc

A comma(,) can also be used instead of '#' for specifying a tag. This is done to reduce keystrokes. Eg: hanuma ,sarva

To go to dashama skandha of bhaagavata, you can type '10.1.1 #bhaaga'
